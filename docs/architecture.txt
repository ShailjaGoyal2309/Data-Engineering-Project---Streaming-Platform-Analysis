# Streaming Analytics Platform Architecture

## Overview

This project demonstrates an end-to-end streaming analytics platform built on Microsoft Azure using a Medallion Architecture (Bronze, Silver, Gold). The system simulates streaming data, automatically processes it using Azure Data Factory and Azure Databricks (PySpark), stores the processed data in Azure Data Lake Storage Gen2, and visualizes business insights using Power BI.

---

## Components

### 1. Data Generation
- Simulates streaming platform activity.
- Generates:
  - Users
  - Movies
  - Watch Logs
  - Subscription Data

---

### 2. Azure Data Lake Storage Gen2
Stores the data across multiple layers:

- Raw
- Bronze
- Silver
- Gold

---

### 3. Azure Data Factory
Responsible for:

- Storage Event Trigger
- Pipeline orchestration
- Notebook execution
- Automated workflow

---

### 4. Azure Databricks (PySpark)

Processes data through the Medallion Architecture.

#### Bronze Layer
- Raw ingestion
- Minimal transformations

#### Silver Layer
- Data cleaning
- Duplicate removal
- Schema validation
- Feature engineering

#### Gold Layer
- Business aggregations
- Analytics tables
- Reporting datasets

Generated Gold Tables:

- watch_completion_fact
- movie_popularity
- subscription_summary
- trending_genres
- user_genre_preference_matrix

---

### 5. Power BI

Visualizes insights using four dashboard pages:

- Overview Dashboard
- User Analytics
- Content Analytics
- Behavior Insights

Includes:

- KPI Cards
- Line Charts
- Bar Charts
- Donut Charts
- Scatter Plot
- Matrix Heatmap
- Interactive Slicers

## Architecture Pattern

This project follows the Medallion Architecture.

Raw Layer
↓

Bronze Layer
↓

Silver Layer
↓

Gold Layer
↓

Analytics

This architecture improves scalability, maintainability, and data quality while supporting automated analytics pipelines.
